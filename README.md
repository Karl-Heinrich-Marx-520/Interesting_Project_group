# Interesting_Project_group
Every interesting project I generated are all placed in this repository，which is quite suitable for students to learn from.

## 异步日志打印系统设计与实现
基于 C++17 实现的**可拓展、非阻塞、线程安全**异步日志系统，支持格式化占位符、多线程并发写入、后台异步落盘，完美解决主线程日志打印阻塞问题。

## 编译期多态实现任务管理系统 (CRTD)
集成 CRTD模式、单例模式、实现的**高性能、持久化、线程安全**命令行界面的任务管理系统，允许用户添加、删除、查看和管理他们的待办事项
