## 项目简介
本项目旨在**设计并实现一个高效的异步日志打印系统**，深度应用 C++ 现代特性：
- 可变参数模板（Variadic Templates）
- 多线程并发编程
- 模板折叠（Template Folding）
- 线程安全队列
- 后台异步写入线程

系统提供**友好的格式化日志接口**，日志记录操作完全异步化，**不会阻塞主线程执行**，适合服务端/客户端程序、高性能应用、嵌入式设备等场景使用。

> 适合学习：并发编程、模板编程、异步设计、C++ 工程化实践

## 项目目标
1. 掌握 **C++ 可变参数模板**，实现灵活通用的日志记录接口
2. 深入理解 **多线程与并发**，实现线程安全的日志队列
3. 使用 **模板折叠** 高效处理参数包，实现占位符格式化
4. 实现 **完整异步日志机制**，后台线程独立写入文件
5. 完善 **错误处理与资源管理**，保证系统稳定与优雅退出

## 功能需求
### 1. 日志记录接口
- 支持 `{}` 占位符格式化：`log("Hello {}", name);`
- 支持无占位符自动拼接参数
- 支持任意类型参数：`int/string/double/char*` 等

### 2. 线程安全日志队列
- 多生产者（主线程）+ 单消费者（后台写入线程）模型
- 高效线程安全访问，无数据竞争

### 3. 后台写入线程
- 独立后台线程持续消费日志队列
- 程序退出前**保证所有日志落盘**

### 4. 格式化功能
- 自动替换 `{}` 占位符
- 容错处理：参数不匹配时保留占位符 / 拼接多余参数

### 5. 错误处理
- 捕获文件打开失败、格式化异常等错误
- 系统不崩溃，输出友好异常信息

## 核心特性
✅ **异步非阻塞**：主线程只入队不写文件
✅ **线程安全**：多线程并发日志安全可靠
✅ **格式化占位符**：`log("name: {}, age: {}", name, age)`
✅ **自动类型推导**：支持 int/string/double 等任意参数
✅ **优雅退出**：程序结束前刷完所有日志
✅ **高容错**：参数不匹配不崩溃
✅ **高性能**：低延迟、低锁竞争

## 使用示例
```cpp
// 基础日志
LOG_INFO("程序启动成功");

// 格式化日志
std::string name = "test";
int age = 20;
LOG_INFO("用户信息：姓名={}, 年龄={}", name, age);

// 浮点数
LOG_DEBUG("运行耗时：{} 秒", 1.234);

// 错误日志
LOG_ERROR("文件打开失败：{}", "config.json");
```
